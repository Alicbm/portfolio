Projects

BARGAILIX

I present to you Bargailix, it is an online store that consumes the information through a REST API created just for this application, the REST API connects to a MySQL database with more than 200 products.

Among the features of this app is the option to search for products, filter products by brands, by price ranges, by types of products. You can sort the products by prices, in descending and ascending order.

However, that's not all, you have the option to send products to the shopping cart, discard, add more products or subtract more products from the shopping cart. On the same route you will have access to recommended products for you and more options.

Among the programming languages used to create this app we have Typescript, Node Js, Express Js and React Js.

PELISFLIX

Pelisflix is an application with information of more than 800,000 movies, here you can search for a specific movie, you can filter movies for different categories, you can see the description of each movie, and in the description path you have different options like "The Top Billed Cast ", the related films, a trailer of the film, comments about the film and general information about the film.

This application was built using the REST API, "TheMovieDB". Among the programming languages used to create this application we have Typescript and React Js.
